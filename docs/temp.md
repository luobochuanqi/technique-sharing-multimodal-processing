## 幻灯片设计：实践项目概览（单页）

### 页面定位

分享中“实践环节引入”页，用 60 秒让学生快速了解 GitHub Classroom 作业的内容、要求与收获，激发动手兴趣。

---

### 内容设计

#### 整体布局（卡片式信息分层）

采用 **顶部标题区 + 中部核心流程 + 底部获取区** 的三段式结构，背景延续深色渐变，所有元素以圆角卡片承载。

---

#### 顶部标题区

| 元素           | 内容                                                                 |
| :------------- | :------------------------------------------------------------------- |
| **主标题**     | 🧪 动手实践：多模态 OCR 命令行工具                                   |
| **副标题**     | 串联「附件处理 · OCR · 动态变量」三大模块，GitHub Classroom 自动评分 |
| **右侧标签组** | 三个徽章：`零 API Key` `纯 Python` `即刻运行`                        |

---

#### 中部核心区（左右两列 + 下方任务卡）

**左侧：四步流程图（纵向）**

```
📷 输入图片
   ↓
🔍 PaddleOCR 文字提取
   ↓
📝 动态变量模板替换
   ↓
📋 结构化输出
```

每个步骤旁附极小字标注对应函数名（`extract_text_from_image` 等）。

**右侧：终端预览卡片**

```
$ python main.py receipt.jpg -t receipt_template.txt

收据分析结果：
- 商户名称：星巴克咖啡
- 交易日期：2026-04-17
- 交易金额：58.00 元
- 支付方式：微信支付
```

**下方：三个横向任务卡片（学生需完成的核心函数）**

| 任务一                      | 任务二                   | 任务三                |
| :-------------------------- | :----------------------- | :-------------------- |
| `extract_text_from_image()` | `apply_template()`       | `process_document()`  |
| 调用 PaddleOCR 提取图片文字 | 解析 `{{变量}}` 模板语法 | 整合前两步 + 字段解析 |

每个卡片底部用小字标注对应评分点数量（例如“3 个评分点”）。

---

#### 底部获取区

- 左侧：**GitHub Classroom 邀请二维码**（带扫描光效）
- 中间：短链接 `gh-classroom.com/xxxxx`
- 右侧：两个关键信息图标
  - ✅ **本地运行，无需 GPU**
  - ⚡ **提交即自动评分，即时反馈**

---

### 动画设计

| 元素         | 动画效果                               | 触发时机               |
| :----------- | :------------------------------------- | :--------------------- |
| 主标题       | 由下至上浮入                           | 进入页面即触发         |
| 流程图节点   | 从上至下依次出现，节点之间连线生长动画 | 标题出现后 0.3s 开始   |
| 右侧终端卡片 | 右侧滑入，终端文字逐行打字机效果       | 流程图出现后 0.2s      |
| 三个任务卡片 | 依次从底部弹入，带轻微弹性             | 终端动画结束后依次触发 |
| 底部二维码   | 淡入 + 扫描光效循环                    | 任务卡片出现后 0.3s    |

---

### 交互设计

| 交互元素       | 行为描述                                                                     |
| :------------- | :--------------------------------------------------------------------------- |
| **流程图节点** | 鼠标悬停显示该步骤的技术实现提示（如“使用 `PaddleOCR(use_angle_cls=True)`”） |
| **终端卡片**   | 点击右上角复制图标可复制示例命令到剪贴板                                     |
| **任务卡片**   | 点击卡片可展开显示该函数的 TODO 说明和评分点清单（浮层）                     |
| **二维码区域** | 点击可放大二维码并显示“使用手机扫码加入作业”提示                             |
| **键盘快捷键** | 按下 `D` 键显示项目文件结构树（`multimodal-ocr-assignment/` 目录预览）       |

---

### 视觉规范（Web 实现参考）

- **字体**：标题 `Inter Bold 3rem`，正文 `Inter Regular 1.2rem`，代码 `JetBrains Mono 0.9rem`
- **配色**：
  - 卡片背景：`rgba(30, 41, 59, 0.7)` 半透明模糊
  - 边框：`1px solid rgba(148, 163, 184, 0.2)`
  - 高亮色：`#38bdf8`（流程图箭头）、`#fbbf24`（变量占位符）
- **布局**：CSS Grid 实现响应式，内容最大宽度 1400px，居中显示
