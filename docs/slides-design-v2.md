### 幻灯片 8：OCR 技术代际跃迁 —— 从 外置OCR管道 到 “原生融合”

**内容设计**

- 左中右时间线：
  - 外置OCR管道
    - OCR工具 → 提取文字 → 纯文本模型
    - 先用 Tesseract 或 PaddleOCR (OCR 引擎) 扫描图片里的字，再把这段文字喂给 LLM 做阅读理解。
  - 原生三部走
    - 视觉编码器 (通常是ViT)：负责把一张高清图片，拆解、编码成一系列携带图像特征的数据(视觉Token)
    - 映射层 (Projector)：“翻译官”。把 视觉编码器 输出的 视觉Token，映射到 大语言模型 的词嵌入空间(可以被LLM理解)
    - 大语言模型 (LLM)：它接收经过翻译的 视觉Token 和输入的文字指令，进行理解和推理，最后用文字给出答案。
  - 原生融合
    - 路线一：离散Token化（以“统一语言”实现大一统）
      - 核心理念：将文字、图像、声音等所有信息，都转化为同一种“离散Token”序列。
      - 代表模型：美团 LongCat-Next
    - 路线二：端到端原生（以“全新架构”实现统一）
      - 核心理念：不满足于“翻译”成Token，而是重构底层架构，从**注意力机制**等层面实现统一。
      - 代表模型：商汤 NEO

**动画设计**

- 表格逐行高亮进入

**交互设计**

- 滑动条可拖动，实时对比传统与智能 OCR 效果
- 表格每一行点击后下方展示对应的技术论文引用
