## 全局动画规范

### 入场动画（每页切换时）

- 当前页向左滑出 + 淡出（0.4s, ease: power2.in）
- 新页从右滑入 + 淡入（0.6s, ease: power2.out）
- 3D 场景不随页面切换重置，保持连续性

### 微交互

- 所有可点击元素 hover 时：scale(1.02) + 发光增强（0.2s）
- 代码块 hover 时：行号区域高亮
- 卡片 hover 时：边框颜色变为 #00D4FF

### 页面内动画触发

- 进入页面时自动播放（无需点击）
- 可通过"重播"按钮重看动画
- 动画时长控制在 2-4s，不拖沓

## 性能优化策略

1. **Three.js 场景**：仅封面/章节过渡页/结束页启用，内容页销毁释放 GPU
2. **粒子数量**：控制在 2000 以内，使用 BufferGeometry + Points
3. **动画**：GSAP 优于 CSS animation（更可控、不卡帧）
4. **图片**：WebP 格式，懒加载
5. **字体**：preload 关键字体，避免 FOUT
6. **代码分割**：Three.js 相关组件动态 import
