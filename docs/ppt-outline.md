## 技术要点总结

### 核心技术
1. **多模态附件输入**：支持图片、文档、音频等多格式混合输入
2. **VLM 解析与空间锚定**：视觉语言模型提取 UI 结构和坐标信息
3. **动态变量注入**：AST 解析提示词模板，自动注入用户偏好
4. **交错序列协议**：确保多模态输入同步处理，降低延迟

### 演示亮点
- **Legacy 重构演示**：从旧 CSS 到 Vue 3 组件的完整重构流程
- **代码对比**：直观展示新旧代码差异
- **二维码预留**：为后续在线体验预留入口
